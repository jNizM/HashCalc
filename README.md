Calculate hash from string, hex or file via AutoHotkey


## Features
* Generate hash value from string or file
* Generate Password-Based Key Derivation Function 2 (PBKDF2)
* Secure-Salted function
* Verify hash

### Functions
* MD2, MD4, MD5
* SHA-1
* SHA-256, SHA-384, SHA-512
* HMAC
* PBKDF2
